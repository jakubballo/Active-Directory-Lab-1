# Active Directory Lab

## Ziel

Aufbau einer Windows Server 2022 Domänenumgebung
mit Active Directory, DNS und Windows 11 Client.

## Technologien

- Windows Server 2022
- Active Directory
- DNS
- DHCP (optional)
- Group Policy
- Windows 11
- VirtualBox
