# Active Directory Einrichtung

## Ziel

Einrichtung einer Active-Directory-Domäne für die Benutzer- und Computerverwaltung.

## Installierte Rollen

- Active Directory Domain Services (AD DS)
- DNS Server

## Domäne

firma.local

## Domain Controller

DC01

## Organisationsstruktur

Firma
├── IT
├── Vertrieb
├── Buchhaltung
└── Management

## Durchgeführte Aufgaben

- Installation der AD-DS Rolle
- Erstellung einer neuen Gesamtstruktur
- Konfiguration von DNS
- Erstellung von Organisationseinheiten (OUs)
